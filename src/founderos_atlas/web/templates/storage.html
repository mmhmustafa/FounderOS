Storage — Unclaimed Folders

Deleting a discovery profile removes the profile, but keeps the files it collected — its snapshots, captured configurations and run history. Nothing reads those files afterwards, so they do not affect what Atlas shows you; they simply accumulate. This page is where you can see them and take the space back. Reclaiming moves a folder aside, it never deletes it — each sweep goes into a dated archive folder you can move things back out of.
