Data Retention

Retention removes only old discovery history records — the archived point-in-time snapshots. It never deletes live state, audit records, incidents, plans, user accounts, or credentials. The latest record per scope, and any record an open incident references, is always protected. Review the exact preview below before anything is removed; deleting requires a typed confirmation, and a deletion manifest is written.
