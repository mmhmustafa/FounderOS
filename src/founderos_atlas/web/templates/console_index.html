Atlas never opens a session on its own and never runs a command on your behalf. What you type is yours: it is not filtered, and it is not recorded. Atlas records only that a session happened — operator, device, endpoint, credential reference, and times.

A candidate is a port that answered but not as a web interface; Atlas does not call it verified. A self-signed or expired certificate is reported, never hidden — network equipment usually presents one, and that is not proof of identity. Atlas never sends a credential to a device UI and never suppresses your browser's own TLS warning. Opening a web UI is recorded (device, URL, outcome); what you then type into the device is not.
