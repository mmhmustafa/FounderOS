{#- Atlas shared formatting macros. when(value) — renders an ISO-8601 timestamp in the standard Atlas display format (DD-Mon-YYYY HH:MM) while preserving the precise original timestamp in the datetime attribute and tooltip. Values that are not parseable ISO timestamps render unchanged, so pre-formatted strings pass through safely. -#} {%- macro when(value, fallback='—') -%} {%- if value -%}{{ value | timestamp }}{%- else -%}{{ fallback }}{%- endif -%} {%- endmacro -%} {#- risk_badge(level) — a labelled risk badge; never color alone. -#} {%- macro risk_badge(level) -%} Risk: {{ (level or 'unknown') | title }} {%- endmacro -%} {#- confidence(band, percent) — a labelled confidence statement. -#} {%- macro confidence(band, percent) -%} Confidence: {{ (band | title) if band else 'Unknown' }}{% if percent is not none %} ({{ percent }}%){% endif %} {%- endmacro -%}
